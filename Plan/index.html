Website coding

  • HTML (hypertext markup language) - tags and page structure
  • CSS (cascading style sheets) - how HTML will appear, CSS classes
  • Responsive Web Design - how websites look on computers, tablets, phones, etc.
  • JavaScript - optional features such as menus, search, sorting, and interactivity

UI/UX (user interface and user experience)

  • Wireframe - structure and layout of a webpage
  • Graphic design
  • Fonts
  • Color wheel
  • HTML color code (hex codes)

Tools

  • VS Code - main code editor
  • Git - tracks changes and allows rollback
  • GitHub - stores website files online and manages branches
  • Branches - test changes before moving them to main/production
  • Cloudflare Pages - hosts the website and deploys from GitHub
  • Cloudflare DNS - manages domain records
  • Cloudflare R2 - stores large downloads such as PDFs, ZIP files, ebooks, and videos
  • Microsoft Expression Web - old website editor/reference tool
  • Notepad - simple text editing
  • Paint.net - image editing
  • Create forms
  • Security - passwords, backups, HTTPS, Cloudflare protection, and reCAPTCHA/Turnstile
  • Website Policies - cookie banner if needed, privacy policy, and acceptable use
  • Templates - reusable page layouts, headers, footers, and navigation
  • Domain purchase
  • Domain hosting/DNS provider
  • Common sense - is what I'm creating or using legal? Does it make me or my family look bad?
